FAQ / Example

// new up a turtle so we can draw
Turtle myTurtle = new Turtle("Logo is Awesome", 500, 500);

// show the turtle by calling this
myTurtle.clearScreen();

// or by drawing something - like a square
for(int i=0; i < 4; i++) {
    myTurtle.forward(100);
    myTurtle.left(90);
}

// if you're going to support drawing recursively (for things like fractals)
// then you have to use the .push() and .pop() methods to store the state
function tree(Turtle t, int dist=50, ) {
    // doing this allows the turtle to return to its previous position & heading
    // when this function returns
    t.push();

    if(dist > 4) {
        t.forward(dist);
        t.left(20);

        tree(t, 2 * dist / 3);

        t.right(40);

        tree(t, 2 * dist / 3);
    }

    t.pop();
}

Constructor

Turtle(String title, int height, int width)

The constructor is pretty straight forward. You need only provide a title string along with a height and width for the window. The turtle will be positioned at (0, 0) pointing north, which corresponds to the center of your window pointing up.

Turtle Movement

forward(int distance)

Move the turtle forward the distance specified.

backward(int distance)

Move the turtle backward the distance specified.

left(float degrees)

Rotate the turtle counter-clockwise the number of degrees specified.

right(float degrees)

Rotate the turtle clockwise the number of degrees specified

arc(int radius, float arc)

Draw an arc with the radius: radius through arc degrees. The arc is drawn using the turtle's current position as the center of a circle with radius: radius, the arc begins at the turtle's current heading and proceeds arc degrees.
The turtle's state is not modified by this function.

label(String textToWrite)

Write text to the screen with the top-left corner of the bouding box for the text at the turtle's current position.
The turtle's state is not modified by this function.

setXY(int x, int y)

Set the turtle's position to (x, y).

setHeading(float newHeading)

Set the turtle's heading to newHeading

home()

Return the turtle to (0, 0) with a heading of 0° (north/up)
