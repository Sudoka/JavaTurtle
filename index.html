FAQ / Example

// new up a turtle so we can draw
Turtle myTurtle = new Turtle("Logo is Awesome", 500, 500);

// show the turtle by calling this
myTurtle.clearScreen();

// or by drawing something - like a square
for(int i=0; i < 4; i++) {
    myTurtle.forward(100);
    myTurtle.left(90);
}

// if you're going to support drawing recursively (for things like fractals)
// then you have to use the .push() and .pop() methods to store the state
function tree(Turtle t, int dist=50, ) {
    // doing this allows the turtle to return to its previous position & heading
    // when this function returns
    t.push();

    if(dist > 4) {
        t.forward(dist);
        t.left(20);

        tree(t, 2 * dist / 3);
        t.right(40);
        tree(t, 2 * dist / 3);
    }

    t.pop();
}

Constructor

Turtle()

The default constructor creates a window with a canvas size of (500, 500) and a title of: "Logo Turtle Graphics"

Turtle(String title, int height, int width)

The custom constructor is pretty straight forward. You need only provide a title string along with a height and width for the window. The turtle will be positioned at (0, 0) pointing north, which corresponds to the center of your window pointing up.

Turtle Movement

forward(int d)

Move the turtle d units forward.

backward(int d)

Move the turtle d units backward.

left(float deg)

Rotate the turtle counter-clockwise deg degrees.

right(float deg)

Rotate the turtle clockwise deg degrees.

arc(int radius, float arc)

Draw an arc with the radius: radius through arc degrees. The arc is drawn using the turtle's current position as the center of a circle with radius: radius, the arc begins at the turtle's current heading and proceeds arc degrees.
The turtle's state is not modified by this function.

label(String text)

Draw text to the screen with the top-left corner of the bouding box for the text at the turtle's current position.
The turtle's state is not modified by this function.

setXY(int x, int y)

Set the turtle's position to (x, y).

setHeading(float newHeading)

Set the turtle's heading to newHeading

home()

Return the turtle to (0, 0) with a heading of 0° (north/up)

Turtle State

push()

The push function is used in conjunction with the pop() function, described next, to store state information about the turtle in order to facilitate recursive drawing. You must call push() before pop()—nasty undefined behavior results from calling push() more times than pop() (which probably means it'll crash)

pop()

The counterpart function to push(). It is used to restore the previous turtle state so that drawing can resume from the state the turtle was in before the preceeding function was called.

setPenDown(boolean penDown)

Sets the turtle's pen_down attribute—if pen_down is true then lines are drawn whenever the turtle is moved. To move the turtle without marking the canvas call setPenDown(false).

setPenSize(int diameter)

Used to change the size of the pen used to draw on the canvas. Defaults to 1px.

setPenColor(int paletteColor)

Sets the pen color to the color specified in the palette. See the list of colors predefined by logo. A value of paletteColor which does not correspond to a color defined in the palette is ignored.

setPenRGB(int red, int green, int blue)

Sets the pen to the color denoted by the triplet (red, green, blue). This does not alter any of the colors in the palette, for to change the palette see setPaletteColor

setBackgroundColor(int paletteColor)

Sets the background color to one of the colors defined in the palette. A value of paletteColor which does not correspond to a color defined in the palette is ignored.

setBackgroundRGB(int red, int green, int blue)

Sets the background color directly to the color denoted by the triplet (red , green, blue). This does not alter any of the colors in the palette, for to change the palette see setPaletteColor

setPaletteColor(int colorNum, int red, int green, int blue)

Sets one of the colors in the palette to the color denoted by the triplet (red, green, blue). If colorNum is greater than the size of the palette the new color is added to the end of the palette, extending the palette.

showTurtle(boolean turtleVisible)

Used to determine whether or not the turtle is shown while drawing.

clean()

Clears the canvas but does not effect the turtle's position or heading.

clearScreen()

Clears the canvas and returns the turtle to its inital state.

setWindowMode(Turlte.WindowMode)

Sets the window mode to one of the window modes listed in Turtle.WindowMode

setPenMode(Turtle.PenMode)

Sets the pen to one of the pen modes listed in Turtle.PenMode

WindowModes

  • WINDOW - the turtle continues off the edge of the canvas.
  • FENCE - the turtle stops when it hits the edge of the canvas.
  • WRAP - when the turtle hits one side of the window it wraps back around to the opposite side.

PenModes

  • PAINT - draws with the penColor, the default pen mode.
  • ERASE - draws with the backgroundColor.
  • INVERT - flips the color 180° across the color wheel.
